*************************** HTTPS/SSL  APACHE XAMPP ******************************
1. Open C:/Xampp/Apache

2. Create a folder named "crt" in C:/Xampp/apache directory

3. Add cert.conf and make-cert.bat files in C:/Xampp/apache/crt directory

4. Edit the cert.conf file with notepad ++ 

5. Change all the {{ DOMAIN }} in the file to your domain name
     ***
     For example:  {{ DOMAIN }} changed into ---> websecurity.com
     Save file

6. Run make-cert.bat (Double click on the file)
 
7. Enter the domain name you provided, in our case --> websecurity.com then press Enter
Do not fill in the questions that follows
Ignore them all until it says press any key to continue

8. Next, a new folder will be automatically created in our "crt" folder i.e. websecurity.com folder

9. Open it and run the server.crt file

10. Then click on Install Certificate --> Ok --> Choose Local Machine --> Next --> Choose Place all certificate in the following store
--> Select Trusted Root Certificate Authorities --> Click OK --> Next and Finish

11. Next Open your host file as administrator in Notepad++ --> C:/Windows/System32/drivers/etc/host 

12. Add a new line ----- > 127.0.0.1 websecurity.com

13. Save the file 

14. Next Open C:/xampp/apache/conf/extra/httpd-xampp.conf  or C:/xampp/apache/conf/extra/httpd-vhosts.conf

15. Add the following: 

## -- websecurity.com -- ##

#http
 <VirtualHost *:1234>
     DocumentRoot "C:/xampp/htdocs/web-security"
     ServerName websecurity.com
     ServerAlias *.websecurity.com
 </VirtualHost>
 
#https
 <VirtualHost *:443>
     DocumentRoot "C:/xampp/htdocs/web-security"
     ServerName websecurity.com
     ServerAlias *.websecurity.com
     SSLEngine on
     SSLCertificateFile "crt/websecurity.com/server.crt"
     SSLCertificateKeyFile "crt/websecurity.com/server.key"
 </VirtualHost>

16. Restart Apache

17. Test in your browser

---> HTTP    [[ http://websecurity.com   OR http://websecurity.com:1234 ]]
---> HTTPS  [[ https://websecurity.com OR https://websecurity.com:443 ]]
